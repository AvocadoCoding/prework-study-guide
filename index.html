HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • In the head element information like the character set and viewport can be specified
  • h1,h2,h3 etc. are heading types
  • Commands like break, paragraph, line, list and section can be used to order page content
  • Semantic HTML elements indicate the role of their content, as well as giving sturcture to webpage
